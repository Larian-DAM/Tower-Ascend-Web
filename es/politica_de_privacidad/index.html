Política de Privacidad

Última actualización: Mayo 2026

1. Datos que recopilamos

Para el funcionamiento del sistema de Ranking (Leaderboard) y la gestión de marcas personales, recopilamos los siguientes datos de los archivos de partida (JSON):

  • Nombre de Usuario: El alias que introduces manualmente para identificarte en la tabla de clasificación.
  • Tiempo de Finalización: El tiempo registrado en las partidas marcadas como completadas (`finished: true`), utilizado para determinar tu posición en el ranking.
  • ID de Dispositivo: Un identificador anónimo generado por el sistema para evitar duplicidad de registros y asegurar que cada jugador mantenga su récord personal de forma íntegra.

2. Uso de los datos

Tu nombre de usuario y tu mejor tiempo serán públicos dentro de la web y el juego para que otros jugadores puedan ver la tabla de clasificación. Estos datos se utilizan exclusivamente para fines estadísticos y competitivos dentro de Tower Ascend. No utilizamos esta información para fines publicitarios ni la compartimos con terceros.

3. Almacenamiento

Los datos y métricas de juego se gestionan y almacenan de forma segura a través de Google Firebase. Al enviar tu puntuación al ranking, aceptas que esta información sea procesada en sus servidores.

4. Derechos

Como usuario, tienes derecho a solicitar la eliminación de tu nombre y tus tiempos registrados en nuestra base de datos. Para ello, puedes contactarnos directamente en: towerascend@gmail.com.
